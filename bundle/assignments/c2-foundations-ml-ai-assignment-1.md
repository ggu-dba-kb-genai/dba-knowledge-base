**Course:** Foundations of Machine Learning and AI (C2) · **Type:** Individual ·
**Total Marks:** 50

## Summary

Students perform basic data analysis and build/interpret a **linear regression model**
on **synthetic data they generate themselves**. The brief specifies generating ~300 data
points (one numeric dependent variable; four independent variables — two numeric, one
binary, one four-level categorical yielding three dummy variables) in a domain of the
student's choice, then fitting a regression (e.g., via a spreadsheet analysis toolpak or
any tool/language) and documenting the result. Generative-AI tools are **not permitted**
except for the data-generation and residual-plotting steps explicitly called out in the
brief, and no two submissions should be alike given the synthetic data.

## Deliverables & mark allocation

Submit the generated dataset (CSV/XLSX) plus a report (PDF) covering:

- Generated dataset following the specification — **5**
- Problem definition (brief paragraph) — **2**
- Data description (dependent/independent variables and their types) — **3**
- Exploratory data analysis: box plots on numeric variables; identify outliers — **5**
- Model building: screenshot of the model summary output — **5**
- Model analysis: correlation matrix for numeric variables + comment on multicollinearity — **5**
- Interpret Adjusted R² in the context of the business problem — **1**
- Report the linear regression equation — **2**
- Comment on the magnitude of impact of each independent variable — **5**
- Comment on the statistical significance of each variable — **5**
- Four residual plots, analysed with one–two sentences each — **10**
- Model summary: comment on goodness of fit and propose further approaches — **2**

## Submission guidelines

- Responses in a PDF; raw data in CSV/XLSX. Include the student name in the filename.
- Charts must have labelled axes and a legend; include clear screenshots and explanations.
- Generative-AI use is not allowed except for the two steps noted above; code need not be submitted, but the required screenshots are mandatory.
