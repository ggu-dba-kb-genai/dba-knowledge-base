# Reference

* [Agentic Loop and Design Patterns](agentic-loop-design-patterns.md) - Core design patterns of agentic systems including the observe-reason-act-feedback loop, action spaces, autonomous vs. state machine agents, and specialized agent personas.
* [Fairness Metrics and Bias Types](fairness-metrics-and-bias-types.md) - A detailed guide to algorithmic fairness metrics, types of societal and statistical bias, disparate impact, and the incompatibility of fairness criteria.
* [Hugging Face Transformers Framework](huggingface-transformers-framework.md) - Overview of Hugging Face Transformers design philosophy, three-class architecture pattern (config, model, preprocessor), and key APIs like pipeline and Trainer.
* [Parameter-Efficient Fine-Tuning (PEFT)](parameter-efficient-fine-tuning.md) - An advanced method to adapt pretrained models by training only a tiny fraction of weights, reducing memory overhead and allowing dynamic adapter hotswapping.
