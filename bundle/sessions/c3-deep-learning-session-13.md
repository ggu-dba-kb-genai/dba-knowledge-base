The session, delivered by Professor L. Srinivasa Varadharajan on January 4, 2026, marks the class's return from winter break. It begins with an exhaustive retrospective of the machine learning and deep learning journey, spanning probability, statistics, and [Supervised Learning Foundations](../concepts/supervised-learning-foundations.md) (such as linear regression, logistic regression, confusion matrices, and decision trees) to deep neural network features (backpropagation, SGD, dropouts, transfer learning, CNNs for [Computer Vision](../concepts/computer-vision.md), and autoencoders).

The primary focus is modeling sequential or time-dependent data where observation ordering is crucial. Unlike snapshot datasets where data points are treated as independent and identically distributed (i.e.d.) and split randomly, time series data exhibits internal autocorrelation and temporal lags. The professor presents concrete sequence examples, such as an Australian weather dataset using 22 attributes (minimum/maximum temperature, pan evaporation, rainfall, wind speed/direction, humidity, cloud cover, and pressure) to predict the next day's rain, Amsterdam bike rentals, and textual sequences.

To address sequential dependency, the course introduces Recurrent Neural Networks (RNNs) as a class of [Neural Network Architectures](../concepts/neural-network-architectures.md). Inspired by feedback connections and lateral pathways in biological mammalian brains, RNNs maintain a dynamic internal memory state. Unlike feedforward networks, RNNs share a single set of weight parameters ($U$ for input, $W$ for hidden states, $V$ for outputs) across all time steps, enabling parameter sharing across arbitrary input lengths. The lecture outlines temporal unfolding and Backpropagation Through Time (BPTT). It details the mathematical formulation of hidden state transitions:

$$s_t = \tanh(W s_{t-1} + U x_t + b)$$

And output formulation:

$$o_t = \sigma(V s_t + c)$$

Where $x_t$ is the input vector at time step $t$, $s_t$ is the hidden state vector (representing the cumulative memory of the sequence), and $o_t$ is the output vector. BPTT optimization can suffer from exploding gradients (resolved via gradient clipping) and vanishing gradients (mitigated by $\tanh$ activations or long delay connections).

To mitigate the rapid context decay of vanilla RNNs over sequences longer than 10 steps, the lecture pivots to Gated Recurrent Units, specifically Long Short-Term Memory (LSTM) networks. Through a linguistic context-building example—deducing the ticket booking destination "Paris" from a long anecdote about a French girl named Emily—the professor explains how LSTMs dynamically preserve, modify, or decay historical contexts. LSTMs introduce two parallel pathways: a long-term memory pathway (the cell state $C_t$ acting as a "conveyor belt" at the top of the network) and a short-term memory pathway (the hidden state $h_t$ at the bottom). 

The flow is carefully regulated by three sigmoid-based gates that scale components between $0$ (completely throw away) and $1$ (completely keep):
- **Forget Gate ($f_t$):** Decides what old information to discard from the cell state.
  $$f_t = \sigma(W_f \cdot [h_{t-1}, x_t] + b_f)$$
- **Input Gate ($i_t$):** Decides which new values to update, using a candidate cell state ($\tilde{C}_t$) generated by a $\tanh$ layer.
  $$i_t = \sigma(W_i \cdot [h_{t-1}, x_t] + b_i)$$
  $$\tilde{C}_t = \tanh(W_c \cdot [h_{t-1}, x_t] + b_c)$$
- **Cell State Update ($C_t$):** Combines forgotten historical context and new scaled candidate values.
  $$C_t = f_t * C_{t-1} + i_t * \tilde{C}_t$$
- **Output Gate ($o_t$):** Decides what information to output and pass as the next hidden state.
  $$o_t = \sigma(W_o \cdot [h_{t-1}, x_t] + b_o)$$
  $$h_t = o_t * \tanh(C_t)$$

The session concludes with a discussion of real-world forecasting applications (e.g., blood sugar level spike prediction in wearable medical devices, power grid consumption, and stock market predictions) and framework trade-offs, highlighting PyTorch's current dominance in research and production.

# Key Concepts

- **State Representation and Parameter Sharing** — Unlike feedforward networks, RNNs maintain a dynamic hidden state $s_t$ that builds a cumulative representation of the past. Parameter sharing ensures that the same weight matrices ($U, V, W$) are applied across all timestamps, allowing the model to process sequences of arbitrary lengths.
- **Backpropagation Through Time (BPTT)** — The training mechanism for RNNs where the network is conceptually unfolded across temporal sequences. Gradients are computed at each time instance and then averaged together to update the shared weights.
- **Vanishing and Exploding Gradients** — Optimization issues arising from recurrent matrix multiplications over long sequences. Exploding gradients are resolved via gradient clipping (scaling down large gradients), while vanishing gradients are mitigated using the $\tanh$ activation function (which decays slower than sigmoid) or long-delay connections.
- **Gating Mechanism** — Sigmoid-based continuous filters outputting values in $[0, 1]$. These gates regulate the flow of information by determining how much of the past context should be forgotten, how much current input should be integrated, and what should be outputted.
- **Long Short-Term Memory (LSTM)** — A specialized gated RNN architecture designed to carry context over many steps (over 1000). It separates sequence memory into two parallel streams: the long-term cell state ($C_t$) conveyor belt and the short-term hidden state ($h_t$).
- **Choosing Classical vs. Gated Models** — For structured sequence data, the professor advises using traditional ARMAX methods when historical relationships are straightforward and linear. For complex, highly nonlinear dependencies, deep gated RNNs (LSTMs) are significantly more robust.
